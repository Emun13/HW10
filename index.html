The API I selected was a random cat API. It can be found on apilist.fun and the specific API's name is Randomcat. The options I used was using a GET request and parsing the JSON to get the "file" option, which allowed me to get a variety of images. This API could be useful for large companies. They can take the same idea and create a data base to hold images of the companies to let users check them out. This can also be used for an animal adoption website. Users could click on a button which can show them different pictures of cats and descriptions which will help adopting animals, such as cats, a lot easier.
